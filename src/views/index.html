Request Header Parser Microservice

Retrieve your IP address, language, and software information from the request headers

How It Works

This microservice extracts essential information from the request headers and returns it in a JSON format. The data includes your IP address, preferred language, and software (User-Agent) information.

Simply make a GET request to the specified API endpoint to receive these details in the response.

Example Usage:

  • [project url]/api/whoami

Example Output:

{"ipaddress": "159.20.14.100", "language": "en-US,en;q=0.5", "software": "Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"}

Note:

This microservice is designed for demonstration purposes, showing how to retrieve and process request headers in an API context.

This project provides a simple API for parsing request headers.

Important: This is purely for demonstration and testing purposes on the API level.
